# Demo Projects

This repository contains several demo projects where I experimented with different technologies. These projects were built to try out ideas, learn frameworks, and practice real-world concepts.

---

## Projects

### 1. Vue Demo (`vue-demo`)
- **Tech:** Vue.js + TypeScript
- **Description:** A one-page scrolling site built while learning Vue and TypeScript. Demonstrates component structure, event handling, and smooth scrolling techniques.
- **Folder:** `vue-demo/`

### 2. React Redux Demo (`redux-demo`)
- **Tech:** React + Redux
- **Description:** A user manager table with features like fetching, filtering, and managing user records. Focused on practicing Redux state management and table interactions.
- **Folder:** `redux-demo/`

### 3. RxAngular Demo (`rxAngular-demo`)
- **Tech:** Angular + RxActions
- **Description:** A demo of state management with RxActions. Includes a periodic table where you can edit records and interactively update state.
- **Folder:** `rxAngular-demo/`

---

## How to Run

Each project is self-contained in its folder. To run a project:

```bash
cd <project-folder>
npm install
npm run serve   # or npm start depending on the project
